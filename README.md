# GameFramework-Lite

轻量级 Unity 游戏开发框架，沉淀通用游戏模块，用于快速搭建小型游戏原型。

## 项目定位
个人维护的模块化游戏工具集，覆盖 UI 管理、角色控制、战斗系统、AI 行为等常用模块，
目标是做新项目时可以直接复用基础框架，减少重复开发。

## 模块概览
| 模块 | 状态 | 核心能力 |
|------|------|---------|
| UI 管理 | 进行中 | 弹窗栈管理、对象池、自适应工具 |
| 角色控制 | 待开发 | 状态机、输入抽象、RootMotion 封装 |
| 战斗系统 | 待开发 | 技能系统、伤害计算、Buff 管理 |
| AI 行为 | 待开发 | 有限状态机、行为树封装、NavMesh 封装 |
| 基础工具 | 进行中 | 事件中心、单例基类、资源加载封装 |

## 设计原则
- **模块化**：各模块低耦合，可单独取用
- **易扩展**：基于接口和抽象类设计，新增功能不改动原有代码
- **轻量化**：不引入重型第三方依赖，保持框架简洁

## 目录结构

Assets/
├── 01-UIFramework/ # UI 框架模块
├── 02-CharacterSystem/ # 角色系统
├── 03-CombatSystem/ # 战斗系统
├── 04-AIBehaviour/ # AI 行为模块
├── Common/ # 通用基础工具
└── Scenes/ # 各模块演示场景


## 环境要求
- Unity 2022.3 LTS
- .NET Standard 2.1

## 使用方式
1. 将对应模块文件夹拷贝到你的 Unity 项目中
2. 参考 Scenes 目录下的演示场景了解用法
3. 各模块独立，可按需引入

## 版本规划
- **v0.1**：UI 框架基础版（弹窗管理 + 对象池）
- **v0.2**：角色状态机 + 输入系统
- **v0.3**：战斗技能系统初版
- **v0.4**：AI 行为树模块
